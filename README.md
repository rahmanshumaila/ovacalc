
# 🌸 OvaCalc Menstrual Cycle Tracker with Nutrition Guidance

A simple, user-friendly web application that helps users track their menstrual cycle and understand what foods and wellness habits support each phase. The app focuses on clarity, comfort, and practical guidance rather than complex medical data.

---

## ✨ Features

### 🩸 Period Tracking

* Add period start and end dates
* Predict next cycle (default 28-day cycle, customizable later)
* Automatically divide cycle into phases:

  * Period
  * Follicular
  * Ovulation
  * Luteal

---

### 📅 Calendar & Status

* Simple calendar/timeline view
* Highlights current phase
* Displays countdown: **“Next period in X days”**

---

### 🥗 Healthy Eating Guidance

For each cycle phase:

* ✅ Recommended foods
* ⚠️ Foods to limit or avoid
* 💡 Wellness tips (iron, hydration, protein, rest)

---
